# Generation-of-elementary-Signals
##To understand the generation of elementary signals using Matlab Code. And I also using Python Code too.

Sinusoidal wave
```matlab
t=-pi:0.01:2*pi;
f=1;
w=2*pi*f;
y=sin(w*t);
plot(t,y);
xlabel('time');
ylabel('Amplitude');
title('sinusoidal wave');
grid on;
```
